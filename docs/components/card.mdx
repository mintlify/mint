## Card

### Icon Colors

Adjust the icon color by using the `color` property.

<Card title="Colored Card" icon={duotone("lightbulb")} color="#ca8b04">
  Here is a card with a yellow icon
</Card>

```jsx
<Card title="Colored Card" icon={duotone("lightbulb")} color="#ca8b04">
  Here is a card with a yellow icon
</Card>
```

### Adding a URL

Make a card a navigation item by adding the `href` property like you would on an anchor element.

<Card title="Click on me" icon={duotone("link")} href="/quickstart">
  Clicking on this card brings you to the quickstart page
</Card>

```jsx
<Card title="Click on me" icon={duotone("link")} href="/quickstart">
  Clicking on this card brings you to the quickstart page
</Card>
```

### Props

<ResponseField name="title" required type="string">
  The title of the card
</ResponseField>

<ResponseField name="icon" type="Icon">

A valid [Font Awesome](https://fontawesome.com/icons) icon with one of the following wrappers: `solid()`, `light()`, `regular()`, `thin()`, or `duotone()`

</ResponseField>

<ResponseField name="color" type="string">
  The color of the icon as a hex code
</ResponseField>

<ResponseField name="href" type="string">
  The url that clicking on the card would navigate the user to.
</ResponseField>

<ResponseField name="children" type="MDX or string">
  Supplemental information for the card under the title (e.g. description,
  examples)
</ResponseField>

## CardGroup

The `CardGroup` component lets you group multiple `Card` components together. It's most often used to put multiple cards on the same column.

<CardGroup cols={2}>
  <Card title="First Card" icon={duotone("square-1")} color="#ea5a0c">
    Neque porro quisquam est qui dolorem ipsum quia dolor sit amet
  </Card>
  <Card title="Second Card" icon={duotone("square-2")} color="#0285c7">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit
  </Card>
  <Card title="Third Card" icon={duotone("square-3")} color="#16a34a">
    Ut enim ad minim veniam, quis nostrud exercitation ullamco
  </Card>
  <Card title="Fourth Card" icon={duotone("square-4")} color="#dc2626">
    Excepteur sint occaecat cupidatat non proident
  </Card>
</CardGroup>

```jsx
<CardGroup cols={2}>
  <Card title="First Card" icon={duotone("square-1")} color="#ea5a0c">
    Neque porro quisquam est qui dolorem ipsum quia dolor sit amet

  <Card title="Second Card" icon={duotone("square-2")} color="#0285c7">
```
